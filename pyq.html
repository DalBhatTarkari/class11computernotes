Class 11 Past Year Questions(FAQs)

Q: What are the basic components of a computer system?

A computer system consists of hardware (physical components like CPU, RAM, storage) and software (programs and operating systems that control hardware).

Q: What is the difference between primary memory and secondary memory?

Primary memory (RAM, cache) is temporary and fast, while secondary memory (HDD, SSD) is used for permanent storage.

Q: Convert (1011)₂ to decimal.

(1011)2 = 1 × 23 + 0 × 22 + 1 × 21 + 1 × 20 = 8 + 0 + 2 + 1 = (11)10

Q: What is De Morgan’s Theorem?

De Morgan’s Theorem states: 1. (A . B)’ = A’ + B’ 2. (A + B)’ = A’ . B’

Q: What is system software?

System software refers to programs that manage hardware and provide a platform for other software. Examples include operating systems like Windows, Linux, and macOS.

Q: What is the difference between LAN and WAN?

LAN (Local Area Network) is a network limited to a small area, like an office or home. WAN (Wide Area Network) covers large areas, like the internet.

Q: What is an IP address?

An IP address is a unique numerical label assigned to devices in a network to identify and communicate with them.

Q: What is the difference between HTML and CSS?

HTML (HyperText Markup Language) structures web content, while CSS (Cascading Style Sheets) styles and formats the layout of a webpage.

Q: What is JavaScript used for in web development?

JavaScript is a programming language used to add interactivity, animations, and dynamic content to web pages.

Q: What is a programming language?

A programming language is a set of instructions used to communicate with a computer and develop software applications.

Q: What is the difference between a compiler and an interpreter?

A compiler translates the entire code at once before execution, while an interpreter translates and executes the code line by line.
